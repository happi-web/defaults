TESTING DEFAULTS

Introduction

Browsers apply a set of default styles to every webpage. You may not have thought about this directly, but you have undoubtedly encountered it.

Lesson overview

This section contains a general overview of topics that you will learn in this lesson.

You’ll learn about default browser styles. You’ll learn how to use a CSS reset to remove or change those default styles. What are default styles and where do they come from? As you have worked on projects, you likely observed default styles applied to certain elements, such as larger and bolder headings on h1 elements, and blue, underlined links on a elements.
